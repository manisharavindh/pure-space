# PureSpace Architectural Studio

A high-end, ultra-minimalist architectural portfolio website built with React, Vite, Tailwind CSS, and Framer Motion.

## Features

- **Atmospheric Minimalism**: Clean design with vast white space and elegant typography.
- **Non-Linear Navigation**: Horizontal scroll gallery for projects (desktop) and sophisticated vertical stack (mobile).
- **Floating Micro-Menu**: Glassmorphism dock at the bottom.
- **Custom Cursor**: Interactive cursor that reacts to hover states.
- **The Void**: interactive 'breathe' section.
- **Smooth Animations**: Powered by Framer Motion and Lenis scroll smoothing.

## Tech Stack

- **Framework**: React + Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Scroll Smoothing**: Lenis

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run development server:
    ```bash
    npm run dev
    ```

3.  Build for production:
    ```bash
    npm run build
    ```
